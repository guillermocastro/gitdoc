GIT

Git Training :: Initializing

git init

Initializes the local git repository

git add .

Adds all the files to the commit list

git commit -m "commit id"

Cloning repository

git remote add origin https://github.com/guillermocastro/gitdoc.git

Prepares the local git repository to be cloned to the https://github.com/guillermocastro/gitdoc.git destination

git push --mirror https://github.com/guillermocastro/gitdoc.git

Clone the local repository to https://github.com/guillermocastro/gitdoc.git

git clone https://github.com/guillermocastro/gitdoc.git gitdoc

Clone a copy of https://github.com/guillermocastro/gitdoc.git to gitdoc

git push -u origin master

Push all the tagged files to the repository

git push https://github.com/guillermocastro/gitdoc.git

Pushed the current repository to https://github.com/guillermocastro/gitdoc.git
